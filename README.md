Llanquihue Tour App

Descripción:
Este proyecto es una aplicación en Java que lee información de tours desde un archivo .txt.
Los datos se almacenan en una lista ArrayList, permitiendo mostrarlos en pantalla y filtrarlos según una condición.

Estructura del proyecto:

src/
├── model
│   └── Tour.java
├── data
│   └── GestorDatos.java
├── ui
│   └── Main.java
└── resources
    └── tours.txt

Instrucciones para ejecutar el programa:

1. Abrir el proyecto en IntelliJ IDEA.
2. Verificar que el archivo tours.txt esté dentro de la carpeta resources.
3. Ejecutar la clase Main ubicada en el paquete ui.
4. El programa mostrará todos los tours y luego los filtrados por precio.
